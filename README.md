PingüiCartas ❄️
¡Envía un mensaje especial de la forma más tierna! PingüiCartas es una aplicación web responsiva diseñada para enviar notas de amor, amistad o saludos a través de un simpático pingüino mensajero en el Polo Norte.

✨ Características
Animación Interactiva: Mira cómo el pingüino camina por la nieve para entregar tu sobre al buzón.

Generación de PDF Pro: Descarga tu mensaje en un archivo PDF con un diseño elegante, doble borde y paginación automática si el texto es muy largo.

Modo Noche: El diseño de la página cambia automáticamente según la hora de tu dispositivo.

Responsiva: Funciona perfectamente en ordenadores y teléfonos móviles.

Personalizable: Elige entre varios sellos para cerrar tu carta.
